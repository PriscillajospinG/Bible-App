# Local Model Setup

This project does not commit AI model weights to Git.

Download model:

`gemma-270m-it-Q4_K_M.gguf`

Place it in:

`bible_app/assets/models/`

Expected runtime filename:

`gemma-270m.gguf`

Notes:
- Keep model files local only.
- `.gguf`, `.bin`, and `.pt` files are ignored by `.gitignore`.
